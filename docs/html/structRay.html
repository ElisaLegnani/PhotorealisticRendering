Detailed Description

A ray of light propagating in the space

Parameters
originthe 'Point' where the ray originates
dira 'Vec' to identify the direction of propagation
tminfloating-point parameter identifing the "minimum distance" the ray travels (to be multiplied by 'dir', default 1e-5)
tmaxfloating-point parameter identifing the "maximum distance" the ray travels (to be multiplied by 'dir', default INFINITY)
depthinteger parameter giving the number of reflections on surfaces (default 0)
